#  Анти ИИ Память

Игра на развитие памяти. Запомните порядок фигур и повторите его!

## 🎮 Как играть
1. Нажмите **СТАРТ**
2. Запомните последовательность фигур (шар, квадрат, треугольник, звезда)
3. Повторите порядок, нажимая на кнопки
4. При ошибке уровень повторяется — игра не заканчивается!
5. При правильном ответе переходите на следующий уровень

## ✨ Особенности
- 🌟 3D светящиеся фигуры
- 🔊 Звуковое сопровождение каждой фигуры
- 🎬 Видеоэффекты "ВЕРНО"/"НЕВЕРНО"
- 🌌 Фон звездного неба
- 📱 Адаптация под мобильные устройства
- 💾 Сохранение прогресса
